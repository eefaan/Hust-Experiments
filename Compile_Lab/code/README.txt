1. 本实验实现的mini-c编译器均在MAC OS X系统下编译并测试

2. 各个文件夹中的主要源代码内容如下所示：
Lab1: 可执行程序test通过MAKEFILE文件编译产生，实现了词法语法分析功能，词法语法分析功能的测试代码为test.c
Lab2: 可执行程序test通过MAKEFILE文件编译产生，在Lab1的基础上实现了符号表管理及语义分析功能，语义分析功能的测试代码为test.c
Lab3: 可执行程序test通过MAKEFILE文件编译产生，在Lab2的基础上实现了中间代码及目标代码生成的功能，目标代码生成功能的测试代码为test.c

在OS X系统下，对测试代码进行分析测试的指令均为'./test test.c'
若需要将输出文件重定向为文本文件进行记录，则使用指令'./test test.c > temp.log'
